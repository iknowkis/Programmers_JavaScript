### array의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때, k번째에 있는 수를 구하기

#### 풀이)
- map
- const[a,b,c] = command // 구조분해할당
- filter
- sort
```javascript
    function solution(array,commands) {
        return commands.map(command => {
            const [a,b,c] = command
            const val = array
                .filter((value,idx)=>idx>=a-1&&idx<=b-1)
                .sort((a,b)=>a-b)
            const answer = val[c-1]
            return answer;
        })
    }
```
